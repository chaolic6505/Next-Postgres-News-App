## Get Started

Install required dependencies:

```sh
npm install
```

Start the app:

```sh
npm run dev;

cd stepzen;

stepzen start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser. The page will reload if you make edits.
You will also see any lint errors in the console.
